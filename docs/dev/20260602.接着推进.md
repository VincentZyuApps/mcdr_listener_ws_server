## 当前状态

当前版本是 `0.4.0-beta.5+20260601`。

这个版本相比早期 `0.3.x` 已经完成了这几类整理：

- 目录结构更清晰
- 新增了 `config.py`
- 新增了 `lang/`
- 引入了图片域名白名单
- 命令、事件、消息格式化逻辑做了初步拆分
- 插件 reload 时的 WebSocket 端口占用问题已修复

同时也要明确一点：

- 图片点击这条交互链路，目前为了兼容当前运行环境，已经回退为接近 `0.3.x` 的手写 SNBT / tellraw 方案
- 也就是说，当前“图片点击”功能是以兼容优先，不是完全 RText 化的最终形态
